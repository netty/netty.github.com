Netty/Incubator/Transport/io_uring 0.0.24.Final released

by normanmaurer
on 17-Nov-2023

We are happy to announce the release of Netty/Incubator/Transport/io_uring 0.0.24.Final. This release is a bug fix release.

The most important changes are:

  • Change default value of MAX_MESSAGES_PER_READ to 16 for all Channel implementations (#221)
  • Fix race condition in IOUringEventLoop that prevents shutdown (#227)

For more details please visit our bug tracker

Thank You

Every idea and bug-report counts, and so we thought it is worth mentioning those who helped in this area.

Please report an unintended omission.
