Netty 4.0.0.Beta1 out now

by normanmaurer
on 15-Feb-2013

We are happy to announce the first Beta release of the upcoming Netty 4.0.0 release. This release fixes over 150 issues/tasks since the release of Alpha8. 

We consider the API as stable now so it may be the perfect time for your project to test-drive it.  To make it easier for all of you we updated the New and Noteworthy post to reflect latest API changes.  Also we updated the user guide for Netty 4. 

Please don't hesitate to get in touch with us if you have any problems or think we missed something.

For a complete list of changes please checkout our issue tracker.  To download Netty, as usual, please visit the downloads page or get it via maven like usual.

The Netty Team want to thank again the whole community for the feedback and contributions. This would not be possible without all of you. You make a difference!
